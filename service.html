Обслуживание 1С

Функционал, встроенный в типовые конфигурации ПО "1С:Предприятие", предназначенный для отправки электронной отчетности в контролирующие органы (ФНС, ПФР, ФСС, РосСтат, РПН, ФСРАР, ФТС) непосредственно из программы "1С"

Весь функционал современной системы сдачи отчетов в привычной для вас программе 1С.

Подключение 1С-Отчетности для одного юридического лица или индивидуального предпринимателя в рамках договора 1С:ИТС уровня ПРОФ БЕСПЛАТНО!

Все направления для сдачи отчетности через интернет включены в единый тарифный план.
